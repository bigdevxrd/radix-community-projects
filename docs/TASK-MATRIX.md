# Task Matrix — Community Bounties

Real tasks that need doing. Real XRD rewards. Pick one and get paid.

## How It Works

1. Browse tasks below or at [radixguild.com/bounties](https://radixguild.com/bounties)
2. Claim in Telegram: `/bounty claim <id>` (or `/bounty apply <id>` for tasks >100 XRD)
3. Do the work
4. Submit: `/bounty submit <id> <deliverable_url>`
5. Get verified → get paid

**Platform fee:** 2.5% on task creation. Workers receive 100% of net reward.

## How to Create a Task

Badge holders can create tasks:

**Quick:** `/bounty create <xrd> <title>`
**Guided:** `/bounty` → "Create Task" → follow the wizard (category, difficulty, deadline)

Tasks >100 XRD require applications (`/bounty apply`) instead of instant claims.
