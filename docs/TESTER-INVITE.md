# Tester Invite Messages

## Version 1: DM to Individual Testers

---

Hey! I'm building a Telegram bot for Radix governance — think proposals, voting, and on-chain badges, all from a TG group chat.

I'm looking for a couple of people to test it out before a wider launch. Here's what you'd need:

- A Radix Wallet with a small amount of XRD for gas fees

To get started:

1. Open Telegram and find **@radix_guild_bot** — start a chat and type `/start`
2. Use `/register` to link your Radix wallet
3. Use `/badge` to mint your guild badge (this is your on-chain membership)
4. Try `/propose` to create a test proposal
5. Try `/vote` to vote on an existing one
6. Try `/poll` to run a quick poll

You can also check out the dashboard to see everything in one place:
https://156-67-219-105.sslip.io/guild

The main commands to test are: `/propose`, `/vote`, `/poll`, and `/badge`.
