# Charter TBD Proposals

Ready-to-paste proposals for deciding the charter's open questions.
Each TBD in the Radix Network DAO Charter needs a community decision.
Post these in the Guild governance TG group.

## Governance Rules

### Voting Duration
```
/poll How long should standard proposal voting last? | 48 hours | 72 hours | 7 days | 14 days
```

### Minimum Votes (Quorum)
```
/poll What minimum number of votes should a proposal need? | 3 votes | 10 votes | 25 votes | 50 votes
```

### Pass Threshold
```
/propose Standard proposals pass with simple majority (>50% For). Governance amendments require supermajority (>66%).
```

### Who Can Propose
```
/propose Only Guild badge holders can create proposals. Anyone can view and discuss.
```

## DAO Infrastructure

### Platform Choice
```
/poll Which governance platform should the Guild adopt long-term? | CrumbsUp | Consultation v2 (on-chain) | TG bot only | Hybrid (TG + on-chain)
```

### Discussion Forum
```
/propose Use RadixTalk (Discourse) as the official discussion forum. Proposals are voted in TG, discussed on RT.
```

## Council Structure

### Council Size
```
/poll How many members should the Guild council have? | 3 members | 5 members | 7 members | No council (direct democracy)
```

### Council Term
```
/poll How long should a council term last? | 3 months | 6 months | 1 year
```

## Treasury

### Initial Funding
```
/propose Establish a Guild treasury. Initial contributors pledge 5,000 XRD each. Minimum 3 contributors to launch.
```

### Spending Authority
```
/poll Who approves treasury spending? | Council only | Council proposes + community votes | Any badge holder proposes + community votes
```

### Spending Limits
```
/poll What's the maximum spend per proposal without supermajority? | 5,000 XRD | 10,000 XRD | 25,000 XRD | 50,000 XRD
```

## Operations

### Hosting
```
/propose Big Dev hosts Guild infrastructure ($14/month) until DAO entity formed. Full handover when ready.
```

### Meeting Cadence
```
/poll How often should the Guild hold community calls? | Weekly | Bi-weekly | Monthly | As needed
```

### Reputation Decay
```
/poll Should badge XP decay with inactivity? | Yes, 10% per month | Yes, 5% per month | No decay | Decay only after 6 months inactive
```

## Notes

- Post one proposal at a time to avoid voter fatigue
- Allow each vote to complete before posting the next
- Results should be documented on RadixTalk for the charter record
- Start with the most impactful decisions first (voting duration, quorum, platform)
