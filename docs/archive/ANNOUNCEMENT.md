# Radix Guild — Governance from Telegram

## What Is It?

A badge-gated governance system for the Radix community. Propose ideas, vote on them, track results — all from Telegram where 18K of us already are.

## How It Works

1. Register your Radix wallet with the bot
2. Mint a free Guild member badge
3. Create proposals and vote in Telegram
4. Badge holders only — your badge is your vote

## What's Live

- **TG Bot** — /propose, /poll, /vote, /amend, /temp, and more
- **Badge Manager** — on Radix mainnet, free mint, 5 tier levels
- **Dashboard** — wallet connect, badge display, live proposals
- **Open source** — MIT licensed, everything on GitHub

## Why?

"No-one can make any decision because of the lack of a simple framework."

This is that framework. Simple enough to use in a chat. On-chain enough to be verifiable. Open enough for anyone to build on.

## Badge Tiers

| Tier | Voting Weight |
|------|--------------|
| Member | 1x |
| Contributor | 2x |
| Builder | 3x |
| Steward | 5x |
| Elder | 10x |

XP earned by voting, proposing, completing tasks. Level up through participation, not speculation.
