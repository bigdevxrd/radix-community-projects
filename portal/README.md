# Rad-DAO Portal

Web3 dashboard for Radix community governance.

## Status: Phase 2 — Design

## Pages

- **Dashboard** — Your badges, XP, level, active tasks, DAOs
- **DAOs** — Browse, join, track all registered DAOs
- **Tasks** — Bounty board, my tasks, submit work
- **Grid Game** — Play the game, leaderboard
- **Badge Explorer** — Look up any badge by ID
- **Profile** — Public profile, reputation, history

## Tech Stack

- Vite + React + TypeScript
- @radixdlt/radix-dapp-toolkit
- @radixdlt/babylon-gateway-api-sdk
