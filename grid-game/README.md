# Grid Game

Gamified onboarding and engagement for Radix DAOs.

## Status: Phase 3 — Design

## Concept

Complete tasks, earn dice rolls. Rolls give bonus XP and rare achievement NFTs. Level up your badge from Newcomer to Elder. Free badge on signup.

## Planned Features

- **Daily dice roll** — free roll every day
- **Task completion rolls** — finish work, get bonus rolls
- **XP system** — 5 levels gating real functionality
- **Achievement NFTs** — rare drops on roll-6 (4% chance)
- **Leaderboards** — top contributors and funders

## Dependencies

- Requires Badge Manager (XP stored on badge via update_stats)
