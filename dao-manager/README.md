# DAO Manager

Multi-DAO registry and aggregation layer for Radix.

## Status: Phase 2 — Design

## Planned Features

- **register_dao** — Any project can register their DAO (permissionless)
- **join_dao** — Badge holders join DAOs with one click
- **list_daos** — Browse all registered DAOs with health stats
- **get_my_daos** — See all DAOs you're a member of
- **Cross-DAO proposals** — View and vote on proposals from one dashboard

## Dependencies

- Requires Badge Manager (identity layer)
- Requires Manager Registry (discovery)
